# Ценность для генерации алгоритмов

Когда мы говорим о задаче генерации алгоритма по условию задачи, на ум обычно приходят некие абстрактные шарообразные лошади в вакууме :

хитрые сортировки списков, преобразования массивов, вычисление сложных функций. И это не даёт увидеть очень важное измерение в самой задаче генерации алгоритма, его значительную ценностную составляющую.

 
Возьмём конкретный пример: задачу обслуживания какого-нибудь химического производства.
